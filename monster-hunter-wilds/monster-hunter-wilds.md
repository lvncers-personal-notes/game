### 激化タイプ

基本は属性。
属性が効く相手に対しては特に有効。
攻撃は会心率の補填が可能であれば候補。

| 激化タイプ | 攻撃力 | 会心率 | 属性値 | 切れ味 |
| - | - | - | - | - |
| 攻撃激化タイプ | +10 | -15% | |
| 会心	| -10 | +10% | -10〜+30(※武器種毎に異なる) | -10 |
| 属性 | | -5% | +30〜+80(※武器種毎に異なる) | |

### 復元ボーナス

切れ味２は必須。
これさえあれば武器スキルで「業物」や「達人芸」などを入れる必要なし。
あとは優先度順に、攻撃、属性、会心。
属性は入れたい場合に１。
会心は入れたい場合に１。
